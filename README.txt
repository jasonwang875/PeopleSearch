After downloading the source code, please make sure "Multiple startup projects" been selected at Solution's property.
Step 1: right click the solution name in Visual Studio
Step 2: choose property
step 3: choose "Multiple startup project" in the solution's property's panel
step 4: make sure "Start" Action is selected for the "FrontEnd" Project and "Web_API" Project.


This Solution support:
1. Add/Search/Edit/Remove record
2. Build-in Filter in result grid (supoort logical combination for "Name" column)
3. Name search text box
4. Simple data validation
5. sorting and paging at search result grid
6. Auto seed data when application starts

Technologies used:
1. Javascript Framework: Kendo UI
2. Backend Service: Web API
3. DataBase: Entity Framework + SQL Express
